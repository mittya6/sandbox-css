CSS Only — <details> + <summary>

CSSだけでアコーディオンを実装できますか？
はい、実装できます。HTMLの <details> と <summary> 要素を組み合わせることで、JavaScriptなしでアコーディオンを作成できます。 ブラウザが開閉の状態管理を行うため、キーボード操作やスクリーンリーダーへの アクセシビリティも標準で確保されます。
name 属性を付けると何が変わりますか？
同じ name 値を持つ複数の <details> は、 同時に開けるのが1つだけに制限されます（排他的アコーディオン）。 Chrome 120 / Firefox 130 以降でサポートされています。
開閉アニメーションはどうやって実現しますか？
::details-content 疑似要素に height: 0 → auto のトランジションを指定します。html { interpolate-size: allow-keywords; } を加えることで height: auto へのアニメーションが可能になります。 閉じる方向のアニメーションには content-visibility への allow-discrete トランジションが必要です。
アクセシビリティは確保されていますか？
<details> と <summary> はHTMLの標準要素なので、WAI-ARIAを手動で追加しなくてもスクリーンリーダーや キーボード操作に対応しています。Enter キーや Space キーで開閉が可能です。
最初から開いた状態にするには？
<details open> のように open 属性をHTMLに記述するだけで、ページ読み込み時から展開した状態で表示されます。 このデモの最初の項目がその例です。
